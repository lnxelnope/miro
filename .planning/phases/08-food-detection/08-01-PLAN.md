<task type="auto">
  <name>Task 3: สร้าง ArScanDetectionService ที่เชื่อมกับ ML Kit ObjectDetector</name>
  <files>lib/features/arscan/application/arscan_detection_service.dart</files>
  <action>
สร้าง service ที่:
1. Initialize ML Kit `ObjectDetector` ด้วย `ObjectDetectorOptions(mode: DetectionMode.stream, classifyObjects: true, multipleObjects: true)` — ใช้ base model ไม่ต้อง custom
2. มี method `Future<List<ArScanDetection>> detectFrame(CameraImage image, InputImageRotation rotation)` ที่:
   - แปลง CameraImage → InputImage (ดู pattern จาก VisionProcessor ที่ `lib/features/scanner/services/vision_processor.dart`)
   - เรียก `objectDetector.processImage(inputImage)`
   - map ผลลัพธ์เป็น List<ArScanDetection> พร้อม normalize rect ให้เป็น 0-1 (หารด้วย image width/height)
   - เลือก primary box = food class ที่มี confidence สูงสุด (≥ 0.6)
   - ถ้าไม่มี food ที่ผ่าน threshold → คืน list ว่าง
3. มี throttle logic: ไม่รัน detection ถ้าเวลาจากรอบก่อนยังไม่เกิน 250ms
4. มี `dispose()` สำหรับปิด ObjectDetector

**สำคัญ:** ดูตัวอย่าง InputImage conversion จาก `lib/features/scanner/services/vision_processor.dart` — ใช้ pattern เดียวกัน (NV21 สำหรับ Android, BGRA8888 สำหรับ iOS)
  </action>
  <verify>
- คอมไพล์ผ่าน
- เปิด ARscan screen แล้วดู log ว่ามีผล detection กลับมา (print label/confidence)
- ไม่มี cloud API call (ตรวจจาก network log)
  </verify>
  <done>detection service ทำงานได้ end-to-end: CameraImage → ML Kit → List<ArScanDetection></done>
</task>
