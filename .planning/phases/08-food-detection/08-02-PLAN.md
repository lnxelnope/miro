เพิ่ม ArScanBoundingBoxOverlay เข้าไปใน Stack ของ ARscan screen ตำแหน่งชั้นบนสุด:

```dart
Stack(
  children: [
    // ชั้นล่าง: CameraPreviewView (จาก 07-01)

    // ชั้นกลาง: controls (ปุ่มกลับ ฯลฯ)
    _buildControls(),
    // ชั้นบน: bounding box overlay
    ValueListenableBuilder<ArScanDetection?>(
      valueListenable: controller.primaryDetection,
      builder: (_, detection, __) => ArScanBoundingBoxOverlay(
        primaryDetection: detection,
        state: controller.currentState.value,
      ),
    ),
  ],
)
```

<task type="auto">
  <name>Task 3: ตรวจ coordinate mapping บนหลายอุปกรณ์</name>
  <files>lib/features/arscan/presentation/widgets/arscan_bounding_box_overlay.dart</files>
  <action>
ตรวจสอบว่า normalizedRect (0-1) ถูก map กับ widget size อย่างถูกต้องบนหลาย aspect ratio:
- คำนวณ: `left = normalizedRect.left * widgetWidth`, `top = normalizedRect.top * widgetHeight` ฯลฯ
- ถ้า camera preview มี aspect ratio ไม่ตรงกับ widget (เช่น preview 4:3 ใน container 16:9) ต้อง compensate — ดูว่า CameraPreviewView จัดการ aspect ratio อย่างไร (fit/fill/cover)
- ทดสอบบนอย่างน้อย 2 ขนาดหน้าจอ (เช่น iPhone + Android mid-range)

**Pitfall:** ถ้า camera preview ใช้ AspectRatio widget หรือ FittedBox ค่า offset ของ overlay จะไม่ตรง — ต้อง match กัน
  </action>
  <verify>bounding box ตรงกับอาหารจริงบนอย่างน้อย 2 เครื่อง/ขนาดหน้าจอ</verify>
  <done>coordinate mapping ถูกต้องข้ามอุปกรณ์</done>
