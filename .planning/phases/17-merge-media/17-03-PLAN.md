# Plan 17-03: Create Meal — gallery per sub-ingredient + ImageAnalysis reuse

## Objective

ใน `create_meal_sheet.dart` แถว **sub ingredient** ให้มี **ปุ่มแกลเลอรี/กล้อง** (แทนหรือเสริม `Icons.search` ที่ ~953–966) และใช้ flow **`ImagePickerService` → `ImageAnalysisPreviewScreen` / `GeminiService`** แบบเดียวกับจุดอื่นในแอป ตาม `17-CONTEXT` D-13–D-15

## Context

- **REQ:** ARC2-MEAL-01
- **Depends:** แผน 02 (บันทึก `imagePath` บนโหนดได้)

## Requirements

1. แต่ละแถว sub ingredient มี control เปิดแกลเลอรี/กล้อง
2. หลังได้รูป → วิเคราะห์ด้วย pipeline เดิม (ไม่ fork prompt)
3. ผลลัพธ์ที่เกี่ยวกับชื่อ/ปริมาณ/หน่วย/โภชนาการ — **เสนอให้ผู้ใช้ยืนยัน** ก่อน merge เข้าแถว (ตาม UX ที่แอปใช้กับ `ImageAnalysisPreviewScreen`)
4. บันทึก `imagePath` (และ bbox ถ้ามีจาก flow) ลงโหนด ingredient ที่สอดคล้อง v2
5. สร้าง **`17-MANUAL-CHECKLIST.md`** ในโฟลเดอร์ phase — รายการทดสอบมือสำหรับ success ทั้งเฟส 17 (merge + thumb + create meal)
