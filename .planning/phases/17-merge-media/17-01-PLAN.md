  <action>
สร้าง `lib/features/health/services/merge_food_entries.dart` ที่มีฟังก์ชัน:

```dart
class MergeFoodEntriesResult {
  final String foodName;        // "group_<timestamp>"
  final double calories;        // ผลรวม
  final double protein;         // ผลรวม
  final double carbs;           // ผลรวม
  final double fat;             // ผลรวม
  final String ingredientsJson; // v2 serialized
  final String? imagePath;      // จาก entry แรกที่มีรูป
  final MealType mealType;      // จาก entry ที่ timestamp เร็วสุด
  final DateTime timestamp;     // จาก entry ที่ timestamp เร็วสุด
}

MergeFoodEntriesResult buildMergedEntry(List<FoodEntry> entries)
```

Logic:
1. **เรียงตาม timestamp** จากเก่าไปใหม่
2. **สร้าง mainIngredients** สำหรับ v2: แต่ละ entry →
   - ถ้า entry มี `ingredientsJson` → parse ผ่าน `parseIngredientsJson` → ถ้า v2 ใช้ main/sub ที่มี; ถ้า legacy แปลง → main nodes ทั้งหมดเข้า list
   - ถ้า entry ไม่มี `ingredientsJson` หรือ empty → สร้าง main **หนึ่งรายการ** จาก entry (ชื่อ=foodName, macro จาก entry)
3. **ห้าม sub-sub**: ถ้า main มี sub ที่มี sub ซ้อน → flatten ก่อนรวม
4. **Serialize** เป็น v2 ด้วย `serializeIngredientsV2`
5. **ผลรวม macro** = sum จากทุก entry
6. **imagePath** = entry แรก (sorted) ที่มี imagePath ไม่ null
7. **mealType + timestamp** = จาก entry ที่ timestamp เร็วสุด
8. **foodName** = `"group_${DateTime.now().millisecondsSinceEpoch}"`
  </action>
