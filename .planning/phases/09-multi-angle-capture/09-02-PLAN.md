  <name>Task 1: สร้าง AngleGaugeWidget — visual indicator บอกทิศทางเอียง</name>
  <files>lib/features/arscan/presentation/widgets/angle_gauge_widget.dart</files>
  <action>
สร้าง widget ง่าย ๆ (ไม่ต้องซับซ้อน) ที่แสดงให้ผู้ใช้รู้ว่าต้องเอียงกล้องไปทิศไหน:

**แนวทางแนะนำ: arc indicator**
- แสดง arc เล็ก ๆ ด้านบนของจอ (ใช้ CustomPainter)
- จุดปัจจุบัน = currentAngle (จาก sensor)
- จุดเป้าหมาย = target zone range (สี highlight)
- เมื่อจุดปัจจุบันอยู่ใน zone → เปลี่ยนสีเป็นเขียว
- ข้อความสั้นใต้ arc: "Tilt down" / "Hold steady" / "Tilt up" ตาม direction

**Parameters ที่รับ:**
```dart
AngleGaugeWidget({
  required double currentAngle,    // 0-90
  required AngleZone targetZone,   // top/diagonal/side
  required bool isInZone,          // currentAngle อยู่ใน target zone หรือไม่
})
```

**ตำแหน่ง:** วางที่ top-center ของจอ ใต้ safe area ~20px ให้ไม่บัง status bar
  </action>
  <verify>เอียงกล้อง → gauge แสดงตำแหน่งปัจจุบัน + target zone ถูกต้อง</verify>
  <done>gauge widget แสดงผลชัดเจนและเข้าใจง่าย</done>
</task>

<task type="auto">
  <name>Task 2: สร้าง progress indicator + capture confirmation</name>
  <files>lib/features/arscan/presentation/widgets/multi_angle_capture_overlay.dart</files>
  <action>
สร้าง overlay widget ที่ประกอบด้วย:

**1. Progress indicator (ขวาบน หรือ ขวาล่าง):**
- แสดง 3 วงกลมเล็ก ๆ แทน 3 มุม (top/diagonal/side)
- วงที่ถ่ายแล้ว → สีเขียว + icon check
- วงที่กำลังจะถ่าย (target) → สีเหลือง + pulse animation
- วงที่ยังไม่ถึง → สีเทา
- ข้อความ "1/3", "2/3", "3/3" ใต้ progress dots

**2. Capture confirmation (กลางล่าง, ชั่วคราว ~2 วิ):**
- เมื่อ lastCaptureResult เปลี่ยน (capture สำเร็จ):
  a. แสดงข้อความ "Top angle captured (1/3)" / "Diagonal angle captured (2/3)" / "Side angle captured (3/3)"
  b. ใช้ AnimatedOpacity fade in → stay 1.5s → fade out
  c. ตำแหน่ง: bottom-center เหนือ safe area ไม่บังจานอาหาร

**3. Manual shutter button (กลางล่างสุด):**
- ปุ่มกลม สีขาวโปร่ง ขนาดเล็กกว่าปุ่มถ่ายรูปปกติ (เพราะเป็น fallback)
- onTap → เรียก controller.manualCapture()
- เมื่อ isCapturing == true → disabled (กัน double tap)
  </action>
  <verify>
- เห็น progress dots + ข้อความ "1/3" บนจอ
- ถ่ายรูปสำเร็จ → เห็นข้อความ confirm fade in/out
- ปุ่ม manual ทำงานได้
  </verify>
  <done>UI elements ทำงานครบ</done>
</task>

<task type="auto">
  <name>Task 3: เพิ่ม haptic + เสียง feedback เมื่อ capture สำเร็จ</name>
  <files>lib/features/arscan/presentation/widgets/multi_angle_capture_overlay.dart</files>
  <action>
เมื่อ capture สำเร็จ (lastCaptureResult เปลี่ยน):

1. **Haptic:** เรียก `HapticFeedback.lightImpact()` (เบา ๆ ตาม 09-CONTEXT.md)
2. **เสียง:** ใช้ `SystemSound.play(SystemSoundType.click)` หรือถ้าแอปมี audio service อยู่แล้วให้ใช้เสียงชัตเตอร์เบา ๆ
3. **Animation:** นอกจาก text confirmation แล้ว ให้ progress dot ของมุมที่เพิ่งถ่าย มี scale animation (1.0 → 1.3 → 1.0) ประมาณ 300ms

**เมื่อ capture fail (blur/detection หาย):** ไม่ต้องแสดงอะไร ปล่อยให้ระบบ retry เอง (ตาม 09-CONTEXT.md)
  </action>
  <verify>ถ่ายรูปสำเร็จ → รู้สึก haptic เบา ๆ + ได้ยินเสียง click + เห็น dot ใหญ่ขึ้นแวบนึง</verify>
