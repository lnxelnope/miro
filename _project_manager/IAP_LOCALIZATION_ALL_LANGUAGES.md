## How to add in App Store Connect

For each IAP / Subscription:
1. Go to the product in App Store Connect
2. Scroll to "App Store Localization" (consumables) or "Localization" (subscriptions)
3. Click the blue "+" button to add a language
4. Select the language and paste Display Name + Description from the tables above
5. Save, repeat for all languages and all products
