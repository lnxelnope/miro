# เงื่อนไขการใช้งาน — Miro Cal

**วันที่มีผลบังคับใช้:** 11 กุมภาพันธ์ 2026

## 1. ข้อจำกัดความรับผิดชอบ

- ข้อมูลโภชนาการจาก AI **อาจไม่ถูกต้อง 100%**
- แอปนี้ **ไม่ใช่คำแนะนำทางการแพทย์** — ปรึกษาแพทย์หรือนักโภชนาการเสมอ
- ค่า kcal/macro เป็นค่าประมาณเท่านั้น

## 2. API Key

- ผู้ใช้รับผิดชอบ Gemini API Key ของตัวเอง
- ผู้ใช้รับผิดชอบค่าใช้จ่ายของ Google API (ถ้ามี)
- แอปไม่รับผิดชอบกรณี API Key ถูกนำไปใช้งานโดยไม่ได้รับอนุญาต

## 3. ข้อมูลของผู้ใช้

- ผู้ใช้รับผิดชอบข้อมูลของตัวเอง
- แอปไม่มี backup บน cloud — ถ้าลบแอปข้อมูลจะหาย
- แนะนำให้ export ข้อมูลเป็นประจำ (เมื่อฟีเจอร์พร้อม)

## 4. In-App Purchase

- Pro upgrade เป็นการซื้อครั้งเดียว (non-refundable)
- ใช้ Google Play Billing ตามนโยบายของ Google
- สามารถ restore purchase ได้เมื่อเปลี่ยนเครื่อง

## 5. การใช้งาน

- ห้ามใช้แอปเพื่อวัตถุประสงค์ที่ผิดกฎหมาย
- ห้าม reverse engineer หรือ decompile แอป
- สงวนลิขสิทธิ์ในโค้ดและ UI ของแอป

## 6. การยกเลิกบริการ

- ผู้ใช้สามารถลบแอปได้ทุกเมื่อ
- การลบแอปจะทำให้ข้อมูลทั้งหมดถูกลบ (ไม่มี backup)
