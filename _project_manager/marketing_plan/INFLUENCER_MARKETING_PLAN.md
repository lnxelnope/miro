## 0. Executive Summary — ทำไมต้อง Influencer ตอนนี้

Google Ads (UAC + Search) **หาคนที่ "กำลังหา" อยู่แล้ว** — แต่คนไทยส่วนใหญ่
**ยังไม่รู้ว่ามีแอปที่สแกน AR อาหารได้** จึงไม่ค้นหามัน

Influencer ทำสิ่งที่ Ads ทำไม่ได้:
- สร้าง **"moment ว้าว"** เมื่อคนเห็นเพื่อนเล็งกล้องแล้วเห็นแคลอรี่ลอยบนอาหาร
- ให้ **social proof** (คนจริงใช้จริง ไม่ใช่แบนเนอร์)
- สร้าง **search demand ใหม่** — คนเห็นคลิป → ไป Google "ArCal" → Ads รับลูกต่อ

**กลยุทธ์หลัก: 1 เงิน ได้ 3 งาน**
```
฿30,000
   ├── งาน 1: Organic reach จาก feed ครีเอเตอร์ (ฟรี)
   ├── งาน 2: Repurpose เป็น Google Ads / TikTok Ads creative (ลด cost ทำ creative)
   └── งาน 3: ใส่ในหน้า landing page / Play Store เป็น social proof
```

## 1. หลักคิด — "Minimum Viable Virality"

> งบ ฿30k ไม่มากพอจะ "ซื้อ" awareness ได้ทั้งประเทศ
> แต่มากพอที่จะ "จุดไฟ" ให้คนแชร์ต่อเองได้ — **ถ้าคอนเทนต์ทำให้ถูกจุด**

### จุดที่ ArCal "น่าแชร์" โดยธรรมชาติ (ใช้จุดนี้เป็นหัวใจ):

| ปัจจัย | ทำไมคนอยากแชร์ |
|--------|-----------------|
| **AR สแกนอาหาร** | ภาพ AR bounding box บนอาหารจริง = "เห็นแล้วต้องส่งต่อ" เหมือนตอน Pokémon Go |
| **อาหารไทย = identity** | "แอปนี้รู้จักผัดกะเพรา!" — trigger ความภูมิใจ → แชร์ |
| **ผลลัพธ์ทันที** | เปิด → สแกน → เห็นแคล ใน 10 วิ = เป็นคลิปได้โดยไม่ต้องตัดต่อ |
| **ไม่ต้อง login** | ลดแรงต้านเป็นศูนย์ — ครีเอเตอร์บอก "โหลดแล้วเปิดเลย" ได้จริง |

**หลักการ:** brief ครีเอเตอร์ให้ **โชว์ moment ที่คนดูจะพูดว่า "ว้าว"**
ไม่ใช่ brief ให้อ่านฟีเจอร์ — ถ้าคนดูว้าวก็แชร์เอง ไม่ต้องจ่ายค่า boost

## 2. งบประมาณ ฿30,000 — แบ่งจ่าย

### แผน A: กระจาย 4–5 คน (แนะนำ — ลดความเสี่ยง)

| หมวด | งบ | หมายเหตุ |
|------|-----|----------|
| **Nano/Micro 4–5 คน** | ฿20,000–24,000 | ค่าจ้าง ฿4,000–6,000/คน (รวม deliverables ทั้งหมด) |
| **อาหาร / prop** | ฿2,000–3,000 | จานไทยที่สีสดถ่ายรูปสวย (ถ้าไม่รวมใน fee) |
| **Boost โพสต์ที่ดีที่สุด** | ฿3,000–5,000 | ใช้หลังรู้แล้วว่าโพสต์ไหนทำงาน (Spark Ads / Partnership Ads) |
| **สำรอง** | ฿1,000–3,000 | เผื่อเจรจาเพิ่ม / ค่า prop |
| **รวม** | **฿30,000** | |

### แผน B: มี "ตัวชูโรง" 1 คน (เสี่ยงกว่า แต่ reach สูงกว่า)

| หมวด | งบ | หมายเหตุ |
|------|-----|----------|
| **Micro 1 คน (30k–80k ฟอล)** | ฿10,000–15,000 | "หัวหมู่" ที่มี authority ในสาย health/food |
| **Nano 2–3 คน** | ฿8,000–12,000 | ฿3,000–4,000/คน |
| **Boost** | ฿3,000–5,000 | |
| **รวม** | **฿30,000** | |

> **คำแนะนำ:** เลือก **แผน A** สำหรับ wave แรก — เพราะ:
> 1. ได้ data ว่า "niche ไหน + tone ไหน" ทำงาน
> 2. ได้ creative 4–5 ชิ้นไปทดสอบใน Ads (แผน B ได้น้อยชิ้น)
> 3. ถ้ามีคลิปไหนดัง → ค่อย reinvest ใน wave 2 กับคนที่ใหญ่กว่า

## 3. เลือกครีเอเตอร์ — "ใคร" สำคัญกว่า "กี่คน"

### 3.1 Niche ที่เหมาะกับ ArCal (เรียงตาม priority)

| Priority | Niche | ทำไม | ตัวอย่างลักษณะ account |
|----------|-------|------|------------------------|
| **1 (ต้องมี)** | **Health / ลดน้ำหนัก / คุมอาหาร** | Intent ใกล้ product ที่สุด — follower = คนที่ "อยากนับแคลอยู่แล้ว" | "กินคลีน", "IF 16:8", "30 วัน challenge", คนแชร์ food diary |
| **2 (ควรมี)** | **Food / รีวิวร้าน / สตรีทฟู้ด** | โชว์ USP "อาหารไทยจริง" ได้เต็มที่ — ยอดดูมักสูง | "กินรอบกรุง", "ร้านเด็ด", "สตรีทฟู้ด" |
| **3 (เสริม)** | **Tech / แอปน่าใช้ / productivity** | เล่า "AR + AI + privacy" ให้คนเข้าใจ — คนกลุ่มนี้แชร์ต่อเก่ง | "แอปที่ใช้ทุกวัน", "tech ง่ายๆ", review gadgets |
| **4 (ถ้าเหลืองบ)** | **Fitness / วิ่ง / โยคะ / ยิม** | ฟอลเยอะแต่ระวัง health claim | "คนรักวิ่ง", "โยคะทุกวัน", PT รีวิว |

### 3.2 เช็กก่อนจ้าง (Checklist)

- [ ] **Engagement rate ≥ 3%** (IG) หรือ **≥ 5%** (TikTok) — คำนวณ: (likes + comments) / followers × 100
- [ ] **คอมเมนต์เป็นคนจริง** (ไม่ใช่แต่ emoji หรือ "น่ารักค่ะ" ซ้ำๆ)
- [ ] **Audience ส่วนใหญ่ในไทย** (ถามขอ insights ได้)
- [ ] **ไม่มีรีวิว supplement/ยาลดน้ำหนัก** ในช่วง 3 เดือนล่าสุด (เสี่ยง health claim ลาม)
- [ ] **เคยรีวิวแอป** มาก่อน → ดียิ่ง (รู้วิธีโชว์หน้าจอ + ลิงก์)
- [ ] **รีล/TikTok ล่าสุด 10 อัน** ยอดดูเฉลี่ย ≥ followers/10 (ถ้าต่ำ = algorithm ไม่ส่ง)

### 3.3 แพลตฟอร์มที่ควรเน้น

| แพลตฟอร์ม | ข้อดี | ข้อเสีย | คำแนะนำ |
|-----------|-------|---------|---------|
| **TikTok** | ยอดดูสูงสุดต่อฟอล / algorithm ส่งให้คนใหม่เห็น / Spark Ads ง่าย | คลิกลิงก์ยาก (ลิงก์ in-video ไม่ได้ ต้องใส่ bio) | **เน้น 60%** ของจำนวนคน |
| **IG Reels** | คนไทยใช้เยอะ / stories ลิงก์ได้ / Partnership Ads | Reach ต่อฟอลต่ำกว่า TikTok | **เน้น 40%** |
| **YouTube Shorts** | SEO ดี — ค้นหาเจอในอนาคต | ไม่เหมาะกับ nano (ยอดดูต่ำ) | ถ้าครีเอเตอร์ cross-post ให้ก็ดี แต่ไม่ต้องจ่ายเพิ่ม |

## 4. Creative Brief — "สิ่งที่ต้องให้ครีเอเตอร์ทุกคน"

### 4.1 Key Message (เลือกมุม 1 ข้อ ต่อ 1 คลิป — อย่ายัดทุกอย่าง)

| มุม | Hook แรก (3 วิ) | เนื้อ (10–25 วิ) | CTA (3 วิ) |
|-----|------------------|-------------------|------------|
| **A. "ว้าว AR"** | "เปิดกล้อง เล็งอาหาร แล้วดู..." | เปิด AR scan → เห็น bounding box + kcal ลอยบนจาน → เปิดหน้าผลลัพธ์ | "ลิงก์ในไบโอ / โหลดฟรี" |
| **B. "ผัดกะเพรา ≠ basil stir fry"** | "แอปฝรั่งบอกผัดกะเพรา 200 แคล... จริงป่ะ?" | สแกนผัดกะเพรา → เห็นแยกวัตถุดิบ (ไก่, น้ำมัน, กะเพรา, ไข่ดาว) + แคลจริงที่สูงกว่า | "แอปไทยที่เข้าใจอาหารไทย" |
| **C. "เปิดปุ๊บใช้ปั๊บ"** | "แอปไหนเปิดมาต้องสมัครก่อน? อันนี้ไม่ต้อง" | เปิด → กดสแกน → เห็นผลทันที (ไม่มี login/quiz) + เน้นว่าข้อมูลอยู่ในเครื่อง | "ลิงก์ในไบโอ" |
| **D. "ทั้งวัน 3 คลิก"** | "บันทึกอาหาร 3 มื้อ ใน 30 วินาที" | เลือกรูปจาก gallery 5 รูป → กดวิเคราะห์ → ทั้งวันเสร็จ | "ไม่ต้องพิมพ์ ไม่ต้องค้นหา" |

### 4.2 กฎเหล็ก (ใส่ใน brief)

```
✅ ต้องทำ:
- โชว์หน้าจอแอปจริง อย่างน้อย 5 วินาที (ไม่ใช่กราฟิก)
- ใช้อาหารไทยที่สีสดใส (ผัดกะเพรา, ส้มตำ, ต้มยำ, แกงเขียวหวาน)
- พูดถึง "ฟรี" / "ไม่ต้อง login" / "ไม่ต้องสมัคร" อย่างน้อย 1 ครั้ง
- ลิงก์ที่ให้ต้องมี UTM (เราเตรียมให้)
- ถ่าย portrait 9:16

❌ ห้ามทำ:
- ห้ามใช้คำว่า "ลดน้ำหนักได้จริง", "ผอมแน่นอน" (health claim)
- ห้ามพูดเกินจริง — ใช้ "ช่วยนับแคล / ประมาณการ" ไม่ใช่ "แม่นยำ 100%"
- ห้ามเทียบแอปอื่นโดยตรง (เช่น "ดีกว่า MyFitnessPal")
- ห้ามลบโพสต์ก่อน 90 วัน
```

### 4.3 สิ่งที่ต้องส่งให้ครีเอเตอร์

1. **ลิงก์โหลดแอป** (Play Store / App Store) + UTM แยกต่อคน
2. **Energy Token ฟรี** (ถ้าแจกได้) — ให้ครีเอเตอร์ลองใช้จริง 2–3 วันก่อนถ่าย
3. **โลโก้ ArCal** (arcal_logo.png) ถ้าจะใส่ end card
4. **Brief** ตามด้านบน (ส่งเป็น PDF 1 หน้า)
5. **ตัวอย่าง flow** — screen recording 30 วิ ที่คุณทำเอง ให้ครีเอเตอร์ดูเป็นแนว

## 6. คำตอบสำคัญ: ควรนำวิดีโอ Influencer ไปใส่ Google Ads ไหม?

### คำตอบ: **ใช่ 100% — นี่คือจุดที่ ฿30k กลายเป็นมูลค่า ฿100k+**

### ทำไม?

| เหตุผล | รายละเอียด |
|--------|------------|
| **1. UGC (User-Generated Content) ทำงานดีกว่า polished ads** | CTR ของ UGC สูงกว่า branded content 2–4 เท่า ในเกือบทุก vertical (Meta 2024 data) |
| **2. ลดต้นทุนผลิต creative** | คุณไม่ต้องจ้างถ่ายวิดีโอสำหรับ Ads แยก — ใช้ของครีเอเตอร์ได้เลย |
| **3. Google Ads (UAC) ต้องการ video asset** | ตาม guide ที่ทำไว้ — UAC ที่มีวิดีโอ perform ดีกว่าไม่มีวิดีโอ 2–3 เท่า |
| **4. "Social proof ในแอด"** | คนเห็นว่า "คนจริงใช้จริง" ไม่ใช่โฆษณาบริษัท → trust สูงขึ้น |

### วิธีนำไปใช้ใน Ads (ทีละแพลตฟอร์ม):

#### A. Google Ads (UAC)
```
1. ขอไฟล์วิดีโอต้นฉบับจากครีเอเตอร์ (ไม่มี watermark)
2. อัปโหลดขึ้น YouTube ของ ArCal (ตั้งเป็น Unlisted)
3. ใน Google Ads → UAC campaign → Ad assets → เพิ่มวิดีโอ
4. Google จะ test วิดีโอ influencer vs วิดีโออื่นๆ อัตโนมัติ
   → ถ้า perform ดี Google จะเพิ่ม spend ให้เอง

⚡ เงื่อนไข: ต้องระบุในสัญญาว่า "อนุญาตให้ใช้ในโฆษณา 90 วัน"
   (บางครีเอเตอร์คิดเพิ่ม ฿1,000–3,000 สำหรับ usage rights)
```

#### B. TikTok Spark Ads (แนะนำมาก!)
```
1. ครีเอเตอร์โพสต์ TikTok ตามปกติ
2. ครีเอเตอร์เปิด "Spark Ads Authorization" ใน TikTok → ส่ง code ให้คุณ
3. คุณเอา code ไปใส่ใน TikTok Ads Manager → สร้าง Spark Ad
4. โฆษณาจะแสดงเป็นโพสต์ของครีเอเตอร์ (ไม่ใช่ของแบรนด์)
   → Trust สูงมาก + engagement นับรวมกับโพสต์เดิม

⚡ ค่า boost: ใช้งบ ฿3,000–5,000 จาก "contingency" ที่แบ่งไว้
⚡ CPV (cost per view) ของ Spark Ads ไทย: ประมาณ ฿0.10–0.30
   → ฿5,000 ≈ 15,000–50,000 views เพิ่มเติม
```

#### C. Meta Partnership Ads (IG)
```
1. ครีเอเตอร์แท็ก ArCal เป็น "Paid partnership" ในโพสต์
2. คุณเห็นโพสต์ใน Meta Ads Manager → กด "Boost" ได้โดยตรง
3. หรือสร้าง campaign ใหม่ → เลือก "Use existing post" → เลือกโพสต์ partnership

⚡ ทำงานคล้าย Spark Ads — แอดแสดงเป็นของครีเอเตอร์
```

### สรุป flow ของวิดีโอ 1 ชิ้น:

```
ครีเอเตอร์ถ่าย (฿4,000–6,000)
     │
     ├── 1. โพสต์บน TikTok/IG ──→ Organic reach (ฟรี)
     │
     ├── 2. Spark Ads / Partnership Ads ──→ Paid reach (฿3,000–5,000 boost)
     │
     ├── 3. อัปโหลด YouTube → Google UAC ──→ Paid installs
     │
     └── 4. Embed ใน landing page ──→ Social proof → conversion
          (อนาคต)

= จ่ายผลิต 1 ครั้ง ใช้ได้ 4 ช่องทาง
```

## 8. ไทม์ไลน์ 6 สัปดาห์

```
สัปดาห์ 1 ──── PREPARE ────────────────────────────────
│  - หาครีเอเตอร์ 8–10 คน (ตาม checklist ข้อ 3.2)
│  - ติดต่อ → เจรจา → เลือก 4–5 คน
│  - เตรียม brief + UTM links + Energy token
│
สัปดาห์ 2 ──── ONBOARD ────────────────────────────────
│  - ส่งแอป + brief + token ให้ครีเอเตอร์
│  - ครีเอเตอร์ใช้แอปจริง 2–3 วัน (สำคัญ! ให้เขารู้สึกจริง)
│  - ครีเอเตอร์ส่ง draft สคริปต์/outline → อนุมัติ
│
สัปดาห์ 3 ──── PRODUCE ────────────────────────────────
│  - ครีเอเตอร์ถ่าย + ตัดต่อ
│  - ส่งให้ review ก่อนโพสต์ (เช็ก: flow ถูกไหม, health claim ไหม)
│
สัปดาห์ 4 ──── LAUNCH (กระจายโพสต์) ────────────────────
│  - โพสต์ กระจาย 3–5 วัน (ไม่กองวันเดียว)
│  - วันจันทร์ + วันพุธ + วันศุกร์ (เว้นระยะ ให้ algorithm ทำงาน)
│  - เก็บ Spark Ads code ทันทีที่โพสต์
│
สัปดาห์ 5 ──── AMPLIFY ──────────────────────────────────
│  - ดูตัวเลข → เลือก 1–2 โพสต์ที่ดีที่สุด
│  - Boost ด้วย Spark Ads / Partnership Ads (฿3,000–5,000)
│  - อัปโหลดวิดีโอที่ดีที่สุดขึ้น YouTube → ใส่ใน Google UAC
│
สัปดาห์ 6 ──── MEASURE & DECIDE ─────────────────────────
   - สรุปผลรวม (ดูหัวข้อ 9)
   - ตัดสินใจ: ทำ wave 2 / ขยายงบ / เปลี่ยน niche
```

## 9. วัดผล — KPI สำหรับ Influencer (ต่างจาก Google Ads)

### 9.1 ตัวเลขจากครีเอเตอร์ (ขอ insights)

| Metric | เป้าที่ดี (nano 5k–15k ฟอล) | ดูยังไง |
|--------|------|---------|
| **Views (รีล/TikTok)** | ≥ 5,000 ต่อคลิป | ดูจาก insights |
| **Engagement rate** | ≥ 5% | (likes + comments + shares + saves) / views |
| **Saves** | ≥ 2% ของ views | saves = คนอยากกลับมาดู = intent สูง |
| **Shares** | ≥ 1% ของ views | shares = viral potential |
| **Link clicks** | ≥ 50 ต่อคน | IG stories / TikTok bio link |

### 9.2 ตัวเลขฝั่งเรา

| Metric | วัดที่ไหน | เป้า |
|--------|----------|------|
| **Installs จาก influencer** | Play Console → UTM | ≥ 200 installs รวมทั้ง wave |
| **Cost per Install (influencer)** | ฿30,000 / installs | < ฿150/install (สูงกว่า Ads แต่ได้ content + trust) |
| **first_open (หลัง wave)** | Firebase Analytics | สูงขึ้นจาก baseline ≥ 30% |
| **ai_analysis rate** | Firebase | ≥ 50% ของ install ใหม่ใช้ AI scan |
| **Brand search volume** | Google Search Console | "ArCal" search เพิ่มขึ้นในช่วง wave |

### 9.3 ตัวเลข "ซ่อน" ที่สำคัญกว่า CPI

> ⚠️ อย่าวัด influencer ด้วย CPI อย่างเดียว — มันไม่แฟร์
> เพราะ influencer ให้คุณค่าที่วัดยาก:

| คุณค่าซ่อน | มูลค่าประมาณ |
|------------|-------------|
| วิดีโอ creative 4–5 ชิ้น (ใช้ใน Ads ต่อ) | ฿15,000–25,000 (ถ้าจ้างทำเอง) |
| Social proof สำหรับ landing page | ยกระดับ conversion ทั้งหมด |
| Brand search demand ใหม่ | Google Ads รับลูก — CPI ใน brand keyword ถูกมาก |
| Trust / credibility | คนเชื่อคนจริงมากกว่าแบนเนอร์ |

**วิธีคิดที่ถูก:**
```
ROI ที่แท้จริง = (installs × LTV) + (มูลค่า creative) + (brand search uplift)
                 ─────────────────────────────────────────────────────────
                              ฿30,000
```

## 10. ข้อผิดพลาดที่ต้องเลี่ยง

| ผิดพลาด | ทำไมเสียเงินเปล่า | ทำแทน |
|---------|-------------------|-------|
| **จ้างคนฟอลเยอะ 1 คน ใช้หมด ฿30k** | ถ้าคลิปไม่ดัง = หมดตัว ไม่มี data เปรียบเทียบ | กระจาย 4–5 คน |
| **Brief ยาว 3 หน้า ระบุทุกคำที่ต้องพูด** | ได้คลิปที่ดูเหมือนอ่านสคริปต์ — คนดูรู้ | Brief สั้น + ให้ครีเอเตอร์ใช้แอปจริง 2 วัน |
| **ไม่ขอไฟล์ต้นฉบับ** | ได้แค่ organic reach ช่องเดียว ใช้ใน Ads ไม่ได้ | ระบุในสัญญา: "ส่งไฟล์ full quality + สิทธิ์ Ads 90 วัน" |
| **ให้ครีเอเตอร์ยัดทุก USP ในคลิปเดียว** | คลิปยาว + วุ่นวาย = skip | 1 คลิป = 1 มุม (AR, อาหารไทย, ไม่ login, etc.) |
| **โพสต์ทุกคนวันเดียวกัน** | ไม่มี sustained visibility | กระจาย 3–5 วัน |
| **ไม่ boost โพสต์ที่ดี** | ปล่อยให้ดีที่สุดตายใน organic | เก็บ ฿3–5k ไว้ boost winner |
| **ไม่ใส่ UTM** | วัดไม่ได้ว่าใครให้ install | เตรียม UTM แยกต่อคน |

## 13. เมื่อ Wave 1 จบ — ตัดสินใจ Wave 2

| สถานการณ์ | ตัดสินใจ |
|-----------|---------|

| **มี clip ที่ Ads CTR สูง** | เพิ่มงบ Google Ads ใส่วิดีโอนั้น → ROI สูงกว่าผลิต creative ใหม่ |
